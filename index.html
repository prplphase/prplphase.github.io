Projetos de Engenharia de Dados

Projetos reais envolvendo ingestão, transformação, modelagem e consumo de dados. Os dashboards representam a camada final de visualização dessas arquiteturas.

Comércio Exterior — Pipeline Analítico

Pipeline de dados desenvolvido para análise de exportações e importações, desde a ingestão dos dados brutos até a camada analítica.

  • Fonte: Dados públicos de comércio exterior
  • Ingestão: Python + Airflow
  • Transformação: dbt (bronze → silver → gold)
  • Armazenamento: PostgreSQL
  • Consumo: Power BI

Arquitetura

Os projetos seguem uma arquitetura ELT, com separação clara entre ingestão, transformação e consumo de dados. 

Fonte de Dados
      ↓
Ingestão (Python / Airflow)
      ↓
Armazenamento (CSV / PostgreSQL)
      ↓
Transformação (dbt)
      ↓
Camada Analítica
      ↓
Consumo (Power BI / Streamlit)
